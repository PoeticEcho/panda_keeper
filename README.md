# 🐼 바오 패밀리 유니버스: 사육사의 기적 (Bao Keeper Ultimate 5.0)

[![GitHub license](https://img.shields.io/badge/license-MIT-green.svg)](https://opensource.org/licenses/MIT)
[![Gamer Rating](https://img.shields.io/badge/Rating-5.0%20%2F%205.0-brightgreen.svg)](#)
[![Zero Dependency](https://img.shields.io/badge/Dependencies-Zero-blue.svg)](#)

에버랜드 바오 패밀리(푸바오, 러바오, 루이바오, 후이바오)의 공식 디자인 실루엣과 사육 고증을 완벽하게 반영한 **오픈소스 1인 개발 웰메이드 감성 힐링 웹 타이쿤 게임**입니다.

외부 이미지나 별도의 오디오 소스 파일 없이, **단 하나의 단일 HTML 파일(`index.html`)**로 정밀한 벡터 그래픽 모핑, 사운드 합성 오케스트라, 화려한 파티클 분수 FX를 완벽히 구현해낸 웹 아키텍처 마스터피스입니다.

---

## ✨ 핵심 시스템 및 고증 포인트 (Key Features)

### 1. 🧬 공식 실루엣 외모 동적 모핑 (Morphology System)
에버랜드 공식 굿즈 라인아트를 기준으로 각 판다 가문의 치명적인 비주얼 특징을 실시간 SVG Path 연산으로 구현했습니다.
* **푸바오(Fubao):** 선택 시 정수리 한가운데 시그니처인 **뾰족한 깃털 머리 뿔**이 돋아납니다.
* **러바오(Lebao):** 동근 귀가 사라지고, 특유의 쫑긋하게 솟은 **각진 세모 귀 부장님 모드**가 활성화됩니다.
* **루이바오 & 후이바오(Twins):** 아기 판다 특유의 **둥글둥글한 타원형 눈 패치**가 적용되며, 아담한 크기로 스케일 다운됩니다.

### 2. 🎵 자연주의 웹 오디오 신디사이저 (Web Audio API Synthesis)
바오 패밀리 팬들과 사육사들의 귀를 사로잡는 청각적 고증이 외부 오디오 파일 없이 실시간 주파수 합성으로 뿜어져 나옵니다.
* **대나무 아작 사운드:** 대나무를 와작와작 부러뜨려 씹는 나무 특유의 거칠고 시원한 임펄스 마찰음 재현.
* **꾸르르륵 행복음:** 판다가 극도로 안정되고 행복할 때 내는 성대의 떨림음("꾸르륵~")을 15Hz 초고속 오디오 주파수 변조(LFO 기법)로 구현.
* **코인 & 차임:** 레조넌스 기법을 활용한 맑고 청아한 동전 짤랑 소리 합성.

### 3. 🎆 도파민 분출 파티클 스프레이 (Visual FX)
* 방사장을 터치하거나 사육 업무(급여, 트레이닝)를 완수할 때, 마우스 좌표와 판다 머리 위로 **그라데이션 핑크 하트와 황금빛 코인 요정들이 분수처럼 터져 나가는** 화려한 파티클 시스템을 구축하여 최상의 타격감과 시각적 피드백을 전달합니다.

### 4. 📖 아침 돌발 생태 서사 및 다이어리 (Narrative Ledger)
* 단순 수치 관리 반복 노동을 넘어, 매일 새로운 아침(`Day`)이 밝을 때마다 사육사의 선택을 시험하는 **돌발 생태 이벤트 팝업**이 발생합니다.
* 사육사의 선택 결과에 따라 스탯이 유기적으로 변동하며, 하단의 **사육 일지 수첩**에 영구히 기록되어 다회차 플레이의 깊이를 더합니다.

### 5. 💾 영속성 세이브 & 무결성 관리 (Local Storage)
* 브라우저를 껐다 켜도 내가 키우던 판다의 성장도, 자금, 날짜 데이터가 `localStorage`에 자동 안전 백업됩니다.
* 사육 일지 탭 내에 **'사육 데이터 완전 초기화' 안전장치 시스템**을 구현하여 언제든 새로운 판다와의 인연을 다시 시작할 수 있습니다.

## 🛠️ 기술 스택 (Tech Stack)

* **Language:** HTML5, CSS3, JavaScript (ES6+)
* **Graphics:** Responsive Inline SVG Semantic Vector Layer
* **Audio:** Native Web Audio API (LFO Frequency Modulation / Synthesis)
* **Framework Dependency:** **Zero (순수 Vanilla 코드로만 작성되어 가볍고 완벽한 최적화 보장)**

## 🚀 깃허브 배포 및 실행 방법 (Deployment)

이 프로젝트는 단 하나의 단일 파일 구조이므로, **GitHub Pages**를 활용해 1분 만에 전 세계에 무료 웹 서비스로 배포할 수 있습니다.

1. 본 레포지토리를 복사(Fork)하거나 코드 전체를 다운로드합니다.
2. 다운로드한 `bao_universe_masterpiece.html` 파일의 이름을 **`index.html`**로 변경합니다.
3. 본인의 GitHub 계정에 새로운 저장소(Repository)를 만들고 `index.html` 파일을 업로드합니다.
4. 저장소 상단의 **`Settings` (설정)** ➔ 왼쪽 메뉴의 **`Pages`** 탭으로 이동합니다.
5. Build and deployment 영역의 Branch를 `None`에서 **`main`** (또는 `master`)으로 변경하고 **`Save`** 버튼을 누릅니다.
6. 약 30초~1분 뒤 상단에 활성화되는 고유 주소(URL)를 통해 스마트폰 및 PC로 즉시 플레이할 수 있습니다!

> 💡 **사육사 꿀팁:** 최신 브라우저 보안 정책(Autoplay Policy)으로 인해, 유저가 화면을 최소 한 번 '터치/클릭'해야 오디오 엔진이 활성화됩니다. 볼륨을 켜고 판다를 먼저 한 번 쓰다듬어 보세요!
